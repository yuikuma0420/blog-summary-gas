# ブログ要約フォーム (Google Apps Script)

---

## 使用技術
- **Googleフォーム**: 入力UI  
- **Google Apps Script**: 処理ロジック  
- **Googleドキュメント**: 出力形式  
- **Googleドライブ**: 保存・管理  

---

## ファイル構成
blog-summary-gas/

├── code.js           # メイン処理 (フォーム送信イベント・ドキュメント生成)

├── appsscript.json   # プロジェクト設定

└── README.md         # プロジェクト概要

---

## セットアップ方法

1. **Googleフォームを作成**  
   以下の入力項目を追加してください。＜＞は回答形式:  
   - 要約作成日（必須）＜日付＞ 
   - ブログタイトル（必須）＜記述式（短文＞
   - ブログURL（必須）＜段落＞
   - 著者名  ＜記述式（短文）＞
   - 投稿日  ＜日付＞
   - 親ジャンル（必須）＜ラジオボタン＞  ※任意のワードを設定
   - 子ジャンル  ＜ラジオボタン＞　※任意のワードを設定
   - キーワード  ＜記述＞
   - 要約  ＜記述＞
   - 感想  ＜記述＞
   - アクション  ＜記述＞
   - 用語説明  ＜記述＞

2. **フォームとスプレッドシートを連携**  
   - フォームの「回答」タブからスプレッドシートを作成  
   - スプレッドシートから「拡張機能 → Apps Script」を開き、本リポジトリの `code.js` と `appsscript.json` を貼り付け  

3. **環境依存の値を設定**  
   - コード内の `YOUR_FOLDER_ID_HERE` を保存先のGoogleドライブフォルダIDに置き換える  

4. **トリガーを設定**  
   - Apps Scriptの「トリガー」設定で `onFormSubmit` を選び、フォーム送信時にコードが実行されるようにする  

---

## 注意事項
- 環境依存の値は `YOUR_FOLDER_ID_HERE` のようにダミー値に置き換えているため、任意の設定値に変更してください
